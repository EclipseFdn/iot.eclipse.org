Live Eclipse IoT demonstrator A greenhouse that reports its status and can be controlled remotely.

The demonstration below features a greenhouse connected to iot.eclipse.org's public MQTT broker via a Raspberry Pi gateway running Eclipse Kura.

The user interface is using MQTT over WebSockets for displaying sensors data available on the broker, as well as to send a command for switching the LED on/off.
