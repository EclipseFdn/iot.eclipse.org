Koneki

Koneki delivers tools easing the development, simulation, testing/debugging and deployment of M2M solutions. The main component of Koneki is an IDE for the Lua programming language.

Paho

The Paho project provides scalable open-source implementations of open and standard messaging protocols.

Mihini

Mihini will deliver an embedded runtime running on top of Linux, exposing high-level Lua API for building M2M applications.

m2m.eclipse.org

m2m.eclipse.org is where you can learn about the technologies developed at Eclipse to make M2M development simpler.

These technologies aim at establishing an open, end-to-end, M2M stack.

Tools

Package a one-stop shop IDE for M2M developers.

We believe that Lua is a language very well tailored for M2M, therefore the first component we deliver is an IDE for Lua development, called Lua Development Tools.
